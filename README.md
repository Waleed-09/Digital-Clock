⏰ Digital Clock
📌 Overview

This is a simple Digital Clock web application built using HTML, CSS, and JavaScript.
It displays the current time in real-time and updates every second automatically.

This project was created to practice JavaScript DOM manipulation and timing functions.

⚙️ How It Works

The project uses JavaScript's Date() object to get the current time and setInterval() to update the clock every second.
